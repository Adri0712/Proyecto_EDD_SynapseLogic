SynapseLogic - Analizador de Resiliencia Neuronal
==================================================

Proyecto de Estructuras de Datos (BPTSP06)
Universidad Metropolitana - Trimestre 2526-3

Integrantes:
- Miguel Gallo
- Adriana Julian
- Samuel Djekki

Descripcion:
El programa modela una red sinaptica neuronal como un grafo dirigido.
Cada neurona es un nodo y cada sinapsis es una arista con peso W = d / (v * k).

Funcionalidades:
1. Cargar red sinaptica desde archivo CSV
2. Cargar diccionario de neurotransmisores desde archivo CSV
3. Visualizar el grafo (nodos arrastrables)
4. Detectar zonas aisladas desde una neurona fuente (BFS)
5. Simular flujo de senal con Dijkstra (ruta mas rapida)
6. Simular deterioro cognitivo (k x 1.2 en todas las sinapsis)
7. Agregar o eliminar neuronas y sinapsis

Archivos CSV de prueba incluidos:
- red_sinaptica.csv       : red de 6 neuronas de ejemplo
- neurotransmisores.csv   : diccionario de 50 neurotransmisores

Como ejecutar:
Abrir en NetBeans y ejecutar el archivo ProyectoEDD_SynapseLogic.java
